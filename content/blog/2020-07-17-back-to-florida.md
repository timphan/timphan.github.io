Practicing my Leaflet and data cleaning skills!

This is a ethnicity breakdown of Census-tract level data of Florida using the 2014-2018 ACS datasets, focusing on CVAP (citizen voting age population) data.

I've layered on a dot-density map of renters and internet accessibility.

I've added 2016 and 2018 election results (using Clinton and Gillum's percentages as a rough benchmark for Democratic Party's "blue")

Future iterations of this map will consolidate and cleanup the missing NA's, I got a suspicion we got some consolidated precincts to clean up.

I'm still re-learning Shiny, would ideally like to make a dashboard from all this to be a bit more visually aesthetic.
